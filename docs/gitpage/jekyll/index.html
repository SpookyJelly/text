지킬

깃허브에 markdown 페이지를 생성하여 등록을 하게 되면, 깃허브 내부적으로 지킬을 통하여 html을 빌드 합니다. 하지만, 깃허브로 전송하기 전에 미리 생성된 html 결과물을 확인하기 위해서는 자신의 컴퓨터에 지킬을 설치하여 테스트를 하여야 합니다.


루비설치

지킬은 ruby 언어로 제작된 정적 웹사이트 빌더 입니다. 먼저 ruby 언어를 설치해야 합니다.
