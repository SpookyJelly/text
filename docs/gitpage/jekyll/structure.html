지킬 구조

이번에는 지킬의 내부 폴더 구조에 대해서 알아 봅니다. 지킬의 내부구조를 통하여 효율적인 정적 웹사이트를 생성할 수 있습니다.

지킬(jekll)의 주요 기능은 마크다운, 텍스트로 구성된 컨덴츠를 레이아웃과 결합 하여보기좋게 정적 HTML 파일을 생성하는 것입니다.

디렉토리 구조

c:\jekyll>tree /f
