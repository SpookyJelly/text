SHA1 해시키는 매우 복잡한 모양의 영어와 숫자로 되어 있습니다.
해시는 40자리의 16진수로 입력하다 실수로 잘못 입력할 가능성이 높습니다.
SHA1 해시키는 매우 큰 숫자이기 때문에 고유 접두사로 간략하게 사용할 수 있는데, 해시의 앞쪽 7자만으로도 중복을 방지하면서 전체 키 값을 사용할 수 있습니다.

해시는 매우 큰 값으로 웬만해서는 앞쪽 숫자 값이 변경되는 경우가 드뭅니다.
물론 전체의 키 값을 다 사용하는 것이 좋겠지만, 앞자리 몇 개만 사용해도 충분합니다.
