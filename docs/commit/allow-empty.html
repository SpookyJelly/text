빈 커밋작성

커밋은 변경된 내용을 깃 저장소에 알려주는 명령어 입니다. 파일의 변경사항이 있어야만 커밋할 내용이 생기게 됩니다.

하지만, 파일의 변경내용이 없이도 비어있는 커밋 기록을 할 수 있습니다. 이때 사용하는 명령어는 --allow-emplty 입니다.

$ git commit --allow-empty -m "empty commit"
