단축키

깃허브는 웹 기반의 서비스 사이트입니다.
하지만 브라우저를 통하여 쉽고 빠르게 깃허브를 사용할 수 있도록 단축키를 지원합니다.


shift + /

깃허브 페이지에 접속한 후에 shift+/ 키를 눌러 봅니다.
그림과 같이 단축키 화면이 실행되는 것을 볼 수 있습니다.
