Star

저장소 우측 상단에는 3개의 버튼이 항상 출력됩니다.


인기있는 저장소

Star는 저장소의 인기를 측정할 수 있는 수치입니다.
Star가 많을수록 깃허브에서 인기 있는 저장소로 주목받습니다.

프로젝트의 감사 의미로 Star를 표시하기도 합니다.
자신이 누른 Star의 저장소는 https://github.com/stars에서 확인할 수 있습니다.
