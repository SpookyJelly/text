# diff 내용을 추가하여 커밋
---
커밋 메시지를 작성할 때 `-v` 옵션을 같이 사용하면 vi 에디터에서 diff 내용을 추가할 수 있습니다.  

```
infoh@hojin MINGW64 /e/gitstudy04 (master)
$ git commit -v ☜ diff 내용 추가
```
