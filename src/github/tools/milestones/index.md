# 마일스톤(milestones)
---
깃허브에서는 이슈 기능과 유사한 마일스톤 기능을 추가로 제공합니다. 마일스톤은 이슈 기능을 좀 더 프로젝트 관리 차원에서 한층 더 강화한 기능으로 보면 됩니다.

<br>

## 프로젝트 관리
---
마일스톤을 이용하여 이슈의 그룹화 작업을 할 수 있습니다. 이를 통하여 프로젝트의 진행 사항을 체크합니다.
