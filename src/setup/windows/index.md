# 깃 윈도우 설치
---
많은 사람들이 기본 운영체제로 사용하고 있는 윈도우 환경을 먼저 알아 보도록 하겠습니다. 윈도우 환경에서 깃을 설치하는 방법은 매우 간단합니다. 

<br>

## 다운로드
---
설치파일을 다운로드 받아 실행만 하시면 됩니다.

![git_01](./img/git_01.png)
